더 많은 태그 확인 : https://developer.mozilla.org/ko/docs/Web/HTML/Element/link


**태그
1. <header> : 요소 및 탐색에 도움을 주는 콘텐츠, 제목, 로고, 검색 폼 등
2. <footer> : 페이지 하단의 정보, 보통 해당 페이지의 저작권정보, 주소 등을 나타냄
3. <div> : 레이아웃을 만들거나 콘텐츠를 나누는 컨테이너로 사용하는 태그 (division)
4. <section> : 각 문서에서 의미가 있는 구획을 나눌 때 사용
5. <h1> : 구획의 제목, h1부터 h6까지 있으며 숫자가 낮을수록 높다
6. <img> : 문서에 이미지 삽입
    - src: 링크를 연결
    - alt: 이미지를 설명
7. <a> : 목적지로 연결 (url, 특정 페이지, 파일 등..)
    - href: 목적지 경로
    - download: 해당 목적지로 이동 대신 download를 시킬 때
    - target: 페이지를 표시할 위치
        > _self : 현재 위치에서 이동
        > _black : 새 탭으로 이동
8. <br> : 줄바꿈


**경로
절대경로 (/) , 상대경로 (./) , 상위경로 (../)
1. ./파일명 : 해당 파일의 위치에서 경로를 시작
2. ../파일명 : 해당 폴더의 상위 폴더에서 경로를 시작
3. /파일명 : 프로젝트 시작폴더에서 경로를 시작
